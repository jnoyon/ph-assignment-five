Important FAQ

What is Document Object Model?

DOM means Document Object Model which is a programming interface that transform a webpage into an object. This object is understandable to web browser. Using JS Dom we can make webpage interactive and dynamic.

How do you select an element from the DOM?

There are several to to select an element from the Document Object Model. For Example: getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector() & querySelectorAll()

What is event delegation in the context of DOM and why is it useful?

In Javascript, Even delegation is pattern based upon the concept of event bubling. Using event Delegation we can add or remove list of items, dynamically create form etc. It helps developers to reduce code.

How do you manipulate an element's attributes and styles using dom?

There are several to to select an element from the Document Object Model. For Example: getAttribute(), setAttribute(), removeAttribute() are common of them.
